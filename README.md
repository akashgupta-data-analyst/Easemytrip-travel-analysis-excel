# EaseMyTrip Travel Booking Analysis (Excel Project)

## Project Overview

This project focuses on analyzing travel booking data to identify key business insights such as revenue trends, booking patterns, and cancellation behavior.

The goal of this analysis is to transform raw booking data into a structured and interactive Excel dashboard to support data-driven decision-making.

---

## Problem Statement

Travel companies handle large volumes of booking data, making it difficult to track performance metrics like revenue, cancellations, and booking trends.

This project aims to solve this problem by creating a dashboard that provides clear visibility into key KPIs.

---

## Key KPIs

* Total Revenue
* Total Bookings
* Cancellation Rate
* Total Cancellations

---

## Key Insights

* Identified peak booking months indicating seasonal demand trends
* Analyzed cancellation patterns to understand customer behavior
* Compared performance across different travel destinations
* Evaluated booking trends to support pricing and marketing strategies

---

## Dashboard Features

* KPI Cards for quick insights
* Monthly booking trends
* Revenue analysis
* Cancellation analysis
* Destination-wise performance

---
